Quick Record Search

Fetch your records in fraction of seconds..!!

Menu for Quick Search setting

When the amount of data grows larger, searching a particular record becomes a bit difficult. This simple yet smart functionality can be advantageous to search a particular record quickly from a bunch of data. By installing this module, you will find an option under Settings >> Technical >> Actions >> Quick Search Record menu.

Setting of quick search criteria

We have created one record under Settings >> Technical >> Actions >> Quick Search Record menu. By doing this setting user will be able to search Sales order by "Order Reference" field. The system will search by exact value and it will show record in a new window. In this screen, selecting default means this search criteria will available for all users unless they select specific criteria from user preference.

Quick Search Text box visibility

After defining at least one search criteria, quick search box will be visible at the top right corner of the screen. Quick Search Box cannot be visible in the user's screen either default Quick search preference is not set by ERP admin or any Quick Search preference is not set by the user.

Lets have an example

As per we have set conditions for search sale order by order reference, no matter in which screen user is, if they enter an order reference in the Quick search box then matching record will open in new window.

Fetch multiple records in list view

You can find multiple records in list view as well. If you want to find matching records instead of exact name then this setting will be useful for you. If there are chances of fetching multiple records at a time, then tree view is recommended in settings.

Select your own search criteria from user preference

In case of a large company having different departments like sales, production, crm, accounting will cause different quick search criteria for different roles. For example, the product manager will prefer quick search of products by code and accountant will prefer quick search of customers by their email. In this scenario, each user can select their own quick search criteria from the list of criteria configured by ERP admin from the user preferences screen.
